## Sail App

本アプリのバージョンは```Laravel9```。  
Laravel独自の環境構築インターフェース```Laravel Sail```を使って開発。  
Docker環境を使用。  
```Sail```コマンドへの慣れ・実装における曖昧な理解の解消・Laravel独自機能の理解・テスト学習を目的に、CRUDで簡易SNSを作成。
